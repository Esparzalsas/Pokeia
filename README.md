# 🧠 PokéIA

PokéIA es un asistente en consola hecho en Python que permite analizar Pokémon, tipos y enfrentamientos usando lógica de combate basada en tipos, habilidades, STAB y cobertura de movimientos.

---

## ⚠️ Estado del proyecto

Proyecto personal en desarrollo. La lógica puede cambiar con el tiempo.


## ⚠️ IMPORTANTE: dónde ejecutar el programa
    Se tiene que abrir la terminal desde la carpeta llamada pokeia
    Py main.py
